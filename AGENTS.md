# AGENTS.ms

## CONTEXTO DEL PROYECTO

Un chatbot integrado con IA y la API de WhatsApp Business. El objetivo es crear un chatbot para reporte de incidencias en una linea de produccion de una fabrica.

## Stack Tecnologico
- Python
- LangGraph
- LangChain
- Langchain-openai
- jinja2

Se utiliza UV como gestor de paquetes y entornos virtuales.


## Consejos para el ambiente de desarrollo
- Usa `uv add <package>` para agregar dependencias.
- Usa `uv add --dev <package>` para agregar dependencias de desarrollo.
- Usa `uv run <command>` para ejecutar comandos.
